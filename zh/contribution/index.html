# 项目结构

go-doudou由三个包构成：

  • cmd: 负责go-doudou命令行工具和代码生成器
  • framework: 负责REST/gRPC框架
  • toolkit: 负责工具箱

# cmd包

各核心文件和文件夹的说明请参考下面的注释。
